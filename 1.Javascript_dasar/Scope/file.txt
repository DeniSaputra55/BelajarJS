scope
- merupakan area akses sebuah data
- ada dua jenis scrope global scope dan local scope.
- setiap kita membuat function makan akan membuat local scope untuk function tersebut 
- data di global scope bisa diakses dari local scope, namun data di local scope hanya bisa diakses di local scope tersebut atau di scope local dibawahnyanya ( dalam kasus function dalam funcion).
