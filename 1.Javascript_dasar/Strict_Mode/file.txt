strict mode
- saat kita menjalankan javascript secara default kode program kita berjalan dalam mode tidak strict atau istilahnya sloopy mode
- pada eECMAScript 5 diperkenalkan mode strict dimana ketika strict mode dijalanka maka akan merubah beberapa cara kerja di javascript seperti :
- merubah beberapa javascript error dari yang tadinya silent error menjadi throw error(terlihat)
- memperbaiki beberapa kesalahan engine javascript untuk optimasi
- menolak beberapa kode perintah yang kedepannya akan digunakan di eECMAScript

//cara menyalakan strict mode
- untuk menyalakan strict mode, kita bisa manambahkan use strict pada baris awal file javascript kita
- atau bisa juga ditambahkan di awal function kita.
